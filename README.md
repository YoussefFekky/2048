# 2048
 
An AI program takes an initial 4 x 4 board of 2048 as input (starts with a board of 0's if no input is given,) and creates the 2048 tile using DFS. Input example: 
start([[0, 2, 0, 4],
[0, 0, 0, 0],
[0, 0, 8, 0],
[0, 128, 0, 16]]).
